‚úÖ Common Useful Options
Option	Description
-u <url>	Target URL
-o <dir>	Output directory
-l <level>	Crawling level (depth)
--dns	Perform DNS enumeration
--keys	Extract sensitive keys (API, etc.)
--clone	Clone the website locally
--ninja	Randomize headers to avoid WAF detection
--exclude <regex>	Skip URLs matching regex
